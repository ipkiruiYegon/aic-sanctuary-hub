# aic-sanctuary-hub
AIC Sanctuary Hub is a modern web application built with FastAPI and Jinja templates to serve as a centralized digital platform for AIC church officials. It provides a simple, secure, and engaging way to stay updated on church activities, receive timely notifications, and strengthen fellowship through technology.
